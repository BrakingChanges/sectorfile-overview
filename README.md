# VATSSA's Sector File Development Overview Repository

## How to Contribute

Branch Naming Conventions

- feature/
- bug/
- hotfix/
- docs/

## Commit Message Conventions

- <type>([optional scope]): <description> # subject
- type can be feat:, fix:, style:, doc:
